### PyTest-Solidity

[![PyPI Version](https://img.shields.io/pypi/v/pytest-solidity.svg?color=blue)](https://pypi.org/project/pytest-solidity)

A PyTest library plugin for Solidity language.

#### Installation

```
$ pip install pytest-solidity
```
